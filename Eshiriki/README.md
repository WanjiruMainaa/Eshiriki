#PROJECT NAME: Eshiriki


Eshiriki is a collaborative task management system that turns group goals into finished tasks and projects. From secure authentication to real-time task assignment and team chat, Eshiriki centralizes your workflow, ensuring every contributor is; in sync, meeting deadlines, and building together.

FUNCTIONAL REQUIREMENTS
1. Register users into the system
2. Authenticate users during login
3. Create and manage teams
4. Add users to teams
5. Create tasks
6. Assign tasks to team members
7. Set deadlines and priorities for tasks
8. Update task status
9. Allow comments on tasks
10. Generate and display notifications
11. Store and retrieve data from the database
12. Restrict access to protected pages for authenticated users only
